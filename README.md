## Getting Started

### Prerequisites
- HTML5 compatible browser
- Basic knowledge of web development

### Installation

```bash
git clone https://github.com/yourusername/Profile-Card-2.0.git
cd Profile-Card-2.0
```

## Usage

1. Open `index.html` in your browser
2. Customize the profile information in the HTML
3. Modify styles in the CSS file as needed

## Project Structure

```
Profile-Card-2.0/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Customization

Update the following in the HTML:
- Profile image
- Name and title
- Bio/description
- Social links

## Technologies Used

- HTML5
- CSS3
